# 图片压缩工具

这是一个简单易用的在线图片压缩工具，具有苹果风格的界面设计。

## 功能特点

- 支持上传 PNG、JPG 等格式的图片
- 可自定义压缩比例
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小对比
- 支持下载压缩后的图片
- 响应式设计，支持各种设备访问

## 技术实现

- 使用原生 HTML5、CSS3 和 JavaScript 开发
- 采用 Flexbox 布局实现响应式设计
- 使用 FileReader API 处理文件上传和预览
- 使用 Canvas API 进行图片压缩
- 遵循苹果设计风格，注重用户体验

## 使用说明

1. 点击上传区域或拖拽图片到指定区域
2. 选择需要的压缩比例
3. 预览压缩效果
4. 点击下载按钮保存压缩后的图片
